#  AI Chatbot

An AI-powered chatbot built with Python using Natural Language Processing.

##  About The Project
A conversational AI chatbot that understands user intent and responds intelligently across multiple topics including crypto, finance, weather, and general knowledge.

##  Features
- Natural language understanding
- Multiple conversation topics
- Crypto & finance knowledge base
- Sentiment detection
- Conversation history
- Extensible intent system

##  Built With
- Python 3
- NLTK (Natural Language Processing)
- Pattern matching & intent classification

##  Installation
pip install nltk
python ai_chatbot.py
